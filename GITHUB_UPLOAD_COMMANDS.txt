# GitHub上传命令列表
# 以下是将代码上传到GitHub仓库的所有必要命令
# 请在命令提示符(cmd.exe)或PowerShell中逐一执行这些命令

# 1. 首先确保你已经安装了Git
# 下载地址: https://git-scm.com/download/win

# 2. 打开命令提示符或PowerShell，然后切换到项目目录
cd "i:\渗透测试脚本\爬取hackerone的众测url"

# 3. 如果尚未初始化Git仓库，请执行以下命令
git init

# 4. 设置你的Git用户信息（只需设置一次）
git config --global user.name "你的GitHub用户名"
git config --global user.email "你的GitHub邮箱"

# 5. 添加所有文件到暂存区
git add .

# 6. 提交更改
git commit -m "Upload HackerOne crawl scripts"

# 7. 添加远程GitHub仓库
# 注意：已经为你配置好了正确的仓库地址
git remote add origin https://github.com/zyjzyjlaohu/hackerone-domains-crawler

# 如果之前添加过远程仓库，可能需要先移除它
git remote remove origin

# 8. 强制推送到GitHub（解决冲突）
# 执行此命令时，系统会提示你输入GitHub用户名和个人访问令牌
# 请确保你的个人访问令牌具有写入仓库的权限
git push --force origin main

# 如果遇到错误，请尝试以下故障排除步骤：

# 检查远程仓库连接是否正确
git remote -v

# 检查Git配置
git config --list

# 查看当前状态
git status

# 如果你想完全重新开始
# git init --initial-branch=main  # 重新初始化
# git add .
# git commit -m "Initial commit"
# git push --force origin main

# 成功上传后，你可以在浏览器中访问以下地址查看你的仓库：
# https://github.com/zyjzyjlaohu/hackerone-domains-crawler
